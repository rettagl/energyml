 <xs:documentation>The Product Volume data object can be used to report production flows or other parameters. It could also be used to report other characteristics (pressure, temperature, flow rate, concentrations, etc.) associated with a specific wellhead. It utilizes a general hierarchy of: Product Volume Facility (wellhead, separator, flow line, choke, completion ...) Parameter Set (block valve status, reciprocating speed, available room ...) Parameter Flow (production, injection, export, import, gas lift ...) Product (oil, water, gas, CO2, oil-gas, cuttings ...)
